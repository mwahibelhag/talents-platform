💡 منصة مواهب (Talents Platform)

منصة تفاعلية مدعومة بالذكاء الاصطناعي لتوليد المشاريع وتقسيط تكاليفها بنظام (اشترِ الآن وادفع لاحقاً - BNPL). تم بناء هذا التطبيق باستخدام **Streamlit** وعرضه للتحكيم في المسابقة.
رابط التطبيق : https://talents-platform-l28y2nj4sycxlhuzkqu7hu.streamlit.app/
## 🚀 مميزات المنصة
- **صفحة هبوط جذابة:** تعرض رؤية المنصة وقيمتها المضافة للرواد والمبتكرين.
- **توليد المشاريع بـ AI:** شاشة ذكية تتيح للمستخدم كتابة فكرته ليقوم الذكاء الاصطناعي بتحليلها.
- **محاكاة التوليد الذكي:** شاشة تحميل تفاعلية تظهر مراحل بناء الجدوى والهوية والبرمجة.
- **أنظمة تقسيط مرنة (BNPL):** عرض باقات الدفع المتاحة مع إمكانية الدفع التجريبي وتفعيل المشروع مع تأثيرات بصرية تفاعلية.

## 🛠️ التشغيل المحلي (Local Setup)
إذا كنت ترغب في تشغيل المشروع على جهازك الشخصي، اتبع الخطوات التالية:

1. **تحميل المشروع أو عمل Clone:**
   ```bash
   git clone <رابط-المستودع>
   cd talents_platform
   ```

2. **تثبيت المكتبات المطلوبة:**
   ```bash
   pip install -r requirements.txt
   ```

3. **تشغيل التطبيق:**
   ```bash
   streamlit run main.py
   ```

## 📂 هيكلة الملفات
- `main.py`: الملف الرئيسي المحتوي على كود التطبيق وواجهات المستخدم والتنقل.
- `requirements.txt`: يحتوي على المكتبات البرمجية اللازمة لتشغيل التطبيق (`streamlit`).
- `README.md`: ملف التوثيق الحالي لشرح المشروع وإرشادات التشغيل.
